## Store PV/PVC info in a configamp before the backup

Sometime teams exclude the PVCs from the namespace backup but still want to keep track of the information of PVC and PV
that was present in the namespace. Create a blueprint that you invoke from a policy before we backup the namespace. The blueprint will generate a config map in the namespace that contains information about PVC with the PV it is attached to. This information should be in a yaml file.
